# AES-Encryption

This project provides a Python implementation of AES-256-CBC encryption and decryption, fully compatible with the original Node.js version.
It uses secure cryptographic practices such as:

32-byte (256-bit) AES keys

Random IV generation

PKCS7 padding

Hex-encoded output (IV + Ciphertext)

This project is ideal for secure data transmission, backend encryption, and cross-language cryptography.

## Features

✔ AES-256-CBC encryption

✔ AES-256-CBC decryption

✔ Automatic IV generation

✔ PKCS7 padding/unpadding

✔ Key generation helper

✔ Output format: iv:ciphertext (hex)
